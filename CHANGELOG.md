# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-06-17

### Added
- Initial release of the Python-based `agentladle-mcp-reoi` server.
- Built-in REOI valuation model tool (`reoi_valuation_model`).
- Comprehensive multi-stage residual income projections based on managerial financial statements.
- `FastMCP` integration for zero-configuration schema generation from Pydantic models.
- Support for execution via `uvx agentladle-mcp-reoi`.
